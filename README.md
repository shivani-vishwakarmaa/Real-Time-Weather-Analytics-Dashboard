
## Weather Analytics Dashboard (Power BI)

This project demonstrates the development of a real-time weather analytics dashboard in **Microsoft Power BI**, covering the complete data pipeline from API extraction to interactive visualization.
![Weather Analytics Dashboard](https://github.com/shivani-vishwakarmaa/Real-Time-Weather-Analytics-Dashboard/blob/main/Real-Time%20Weather%20Analytics%20%26%20Correlation%20Dashboard.png)

---

### Key Features

#### Data Integration and Processing

* Connected to a live Weather API to fetch real-time data
* Used Power Query to transform and normalize nested JSON into structured tables

#### Data Modeling

* Designed a relational data model to support efficient filtering and dynamic updates
* Enabled seamless interaction across visuals based on user inputs and location selection

#### Visualization and Insights

* Developed a dark-themed dashboard with KPI cards for temperature, humidity, and other metrics
* Built trend analysis charts to monitor weather patterns over time

#### Exploratory Analysis

* Created correlation visuals to analyze relationships between atmospheric variables such as UV Index and Visibility
* Provided deeper insights into environmental conditions

#### UI/UX Design

* Implemented custom slicers for enhanced interactivity
* Applied conditional formatting and icons to improve readability
* Designed a modern, glassmorphism-style interface for a production-ready experience

---

### Tech Stack

* Power BI (Data Modeling and Visualization)
* Power Query (ETL and Data Transformation)
* Weather API (Real-time Data Source)

---

### Outcome

A fully interactive and visually refined dashboard that delivers real-time weather insights and supports exploratory data analysis through intuitive design and dynamic filtering.
